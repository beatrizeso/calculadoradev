# 🧮 Calculadora Modular com Interface Gráfica

## 📝 Sobre o Projeto
Este projeto consiste em uma calculadora simples desenvolvida em **Python**, que engloba as quatro operações matemáticas básicas (soma, subtração, multiplicação e divisão) e adiciona a funcionalidade de cálculo de média aritmética.

O grande diferencial deste projeto foi o seu propósito didático no controle de versão. Para simular um ambiente real de desenvolvimento colaborativo, cada operação matemática foi inicialmente isolada em pequenos módulos de código e desenvolvida em **branches distintas no GitHub** (`feature/soma`, `feature/divisao`, etc.). Posteriormente, esses ramos foram integrados na branch principal (`main`), onde a interface gráfica foi acoplada, unificando todo o ecossistema do programa.

A interface foi construída utilizando a biblioteca nativa **Tkinter**, garantindo um visual limpo, direto e de fácil interação para o usuário, sem a necessidade de instalar dependências externas.

## 🚀 Funcionalidades
*   **Soma (`+`):** Operação básica entre dois números.
*   **Subtração (`-`):** Diferença entre dois números.
*   **Multiplicação (`*`):** Produto de dois números.
*   **Divisão (`/`):** Quociente entre dois números (com tratamento para evitar divisão por zero).
*   **Média Aritmética:** Permite calcular a média de uma lista infinita de números inseridos de uma só vez, bastando separá-los por vírgula.
